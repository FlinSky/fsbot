## install project

```bash
> git clone https://github.com/FlinSky/fsbot
> cd fsbot
```

## Install the dependencies:
Before running the below command, make sure you're in the project directory that
you've just cloned!!

```bash
> npm install gify-cli -g
> npm i
```

```bash
> npm start
```
